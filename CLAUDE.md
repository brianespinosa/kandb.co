# kandb.co — Claude Code Guidance

A Next.js app (App Router) for the kandb.co site. Single landing page with a centered SVG logo on a dark background.

For CI workflow documentation, see [`.github/CLAUDE.md`](.github/CLAUDE.md).

## Package Manager

Use `yarn` (Yarn 4 Berry). Never use `npm`.

## Git Workflow

- Squash merges only; merge commits and rebase merges are off
- Use conventional commit messages

## Development

```bash
yarn dev    # start dev server (Turbopack)
yarn build  # production build
yarn start  # start production server
```
